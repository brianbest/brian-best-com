A personal toolkit of slash commands that make agentic-coding sessions faster and more reliable:

- **Silent health check** — a system check that stays quiet unless something is actually wrong,
  so it's cheap to run constantly.
- **Prompt-looping** — an iterative loop backed by state files and a bash stop hook, letting a
  prompt re-run itself until a condition is met.
- **Eval-gated prompt improvement** — a command that improves a prompt only when the change
  clears an eval bar, following Karpathy's autoresearch methodology.
